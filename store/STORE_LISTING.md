## 단일 목적 진술 (Single Purpose) — "전용 목적 설명" 필드
```
AutoKorea는 해외 웹사이트의 회원가입, 결제, 주소 입력 폼에서 국가 선택 드롭다운을 자동으로 한국(Republic of Korea)으로 채워주는 단일 목적 확장 프로그램입니다.

해외 사이트마다 한국 표기가 'Korea, Republic of', 'South Korea', 'Republic of Korea', 'Korea (South)', '대한민국' 등으로 다양하고, 알파벳 순으로 정렬된 긴 국가 목록에서 매번 찾는 것이 번거로운 한국 사용자의 불편을 해소합니다.

확장은 페이지에 표시된 국가 선택 위젯(네이티브 select, 커스텀 드롭다운, 전화번호 국가코드 위젯)만 식별하여 한국 옵션을 선택하며, 그 외의 페이지 콘텐츠나 사용자 데이터는 다루지 않습니다.
```

## 권한 사유 (Permission Justifications)

**`storage` 사용 근거**
```
사용자가 옵션 페이지와 팝업에서 설정한 (1) 확장의 전체 활성화 여부와 (2) 자동 선택을 비활성화할 도메인 목록을 chrome.storage.sync에 저장하기 위해 storage 권한을 사용합니다.

저장되는 데이터는 위 두 가지 설정값뿐이며, 외부 서버로 전송되지 않고 사용자의 Google 계정 동기화를 통해 본인의 다른 Chrome 브라우저로만 동기화됩니다. 사용자가 의도치 않은 자동 선택을 사이트 단위로 끌 수 있도록 하는 데 storage 권한이 필수적입니다.
```

**호스트 권한 `<all_urls>` 사용 근거**
```
사용자가 회원가입이나 결제를 진행하는 해외 사이트가 사전에 정해져 있지 않으므로, 어떤 도메인의 페이지든 국가 선택 폼을 발견하면 한국으로 채우기 위해 모든 URL(<all_urls>)에 대한 호스트 권한이 필요합니다.

content script는 페이지의 국가 선택 위젯(<select> 요소, role=combobox 드롭다운, intl-tel-input 등 전화번호 국가코드 위젯)에 해당하는 DOM 요소만 식별하여 'Korea' 옵션을 클릭/선택합니다. 페이지의 다른 내용을 읽거나 가공하지 않으며, 어떠한 데이터도 외부 서버로 전송하지 않습니다. 모든 동작은 사용자의 브라우저 안에서만 처리됩니다.
```

## 원격 코드 사용 중인가요? — "아니요"
모든 JavaScript가 확장 zip 패키지에 포함되어 있고, `eval`, 외부 script 태그, 동적 import, 외부 CDN 로딩 등을 일절 사용하지 않음.

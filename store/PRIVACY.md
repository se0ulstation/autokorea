# AutoKorea 개인정보처리방침

마지막 업데이트: 2026-04-28

## 한 줄 요약
AutoKorea는 사용자의 개인정보를 **수집하지 않고**, **저장하지 않고**, **외부로 전송하지 않습니다**.

## 데이터 처리 내역

| 데이터 | 처리 방식 |
|---|---|
| 방문하는 웹페이지 내용 | 브라우저 안에서만 처리. 외부 서버로 전송 안 함. |
| 사용자 설정 (활성화 여부, 비활성화 도메인 목록) | `chrome.storage.sync`에 저장. Google 계정 동기화를 통해 사용자 본인의 다른 Chrome에만 동기화됨. 개발자에게는 전송되지 않음. |
| 분석/통계/오류 리포트 | **수집 안 함.** |

## 권한 사용 사유

- **`storage`**: 사용자가 옵션 페이지에서 설정한 활성화 여부와 비활성화 도메인 목록을 저장하기 위함.
- **`<all_urls>` (host permissions)**: 사용자가 어떤 사이트에서 회원가입을 하든 국가 선택 폼을 도울 수 있어야 하므로 모든 페이지에서 동작이 필요. 페이지 내용을 외부로 전송하지 않으며, 오직 국가 선택 위젯만 식별·조작합니다.
